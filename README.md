# Ruby Empowers

Pré-requisito: Docker instalado

## Rodando a aplicação

```
docker compose build
docker compose up
```

A aplicação agora pode ser acessada em http://localhost:3000

## Para rodar os testes
```
docker compose run web bin/rspec
```
